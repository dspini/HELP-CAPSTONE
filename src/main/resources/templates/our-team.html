Our Project

The Household Emergency Logistics Planner (HELP) is an application for planning and organizing an emergency response plan for a single household and family groups. HELP is a full stack Java application the allows the user to register and account for personal or groups, create posts and view posts, create checklists, maps and upload documents.


This application utilizes the Mapbox API and Filestack API, HELP is implemented on MySQL database, Spring Boot Security and Thymeleaf, as well as CSS and Bootstrap styling , and Javascript, JQuery, Json, and AJAX.
